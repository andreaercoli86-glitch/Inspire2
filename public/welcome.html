2 Scarica i modelli

Apri un terminale ed esegui questi comandi:

  • ollama pull gemma3:4b — Modello LLM (raccomandazioni)
  • ollama pull nomic-embed-text-v2-moe — Embeddings multilingue

3 Costruisci il database

Il database delle opere non è ancora stato generato. Esegui nel terminale:

  • cd inspire-server
  • npm install
  • npm run build:all

Questo processo richiede circa 30-60 minuti la prima volta.
